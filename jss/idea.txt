Allow users to create JS versions of CSS stylesheets.
For example, consider the CSS stylesheet:

.title
{
	background: white;
	color: black;
	font-size: 20px;
}

div.options
{
	background: black;
	color: white;
	font-family: Georgia, "Times New Roman", serif;
}

This would be written in a JSS stylesheet as:

var stylesheet = 
{
	".title":
	{
		"background": "white",
		"color": "black",
		"font-size": "20px"
	},

	"div.options":
	{
		"background": "black",
		"color": "white",
		"font-family": 'Georgia, "Times New Roman", serif'
	}
}

JSS.apply(stylesheet);
